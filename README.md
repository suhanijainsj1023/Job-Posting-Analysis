## Introduction
This dashboard was created for **Job Seekers, Career Transitioners, and Workforce Analysts** to address a common challenge: information about the data job market is fragmented and difficult to interpret.  

Using a real-world dataset of **2024 data-related job postings** (titles, salaries, benefits, and locations), this project consolidates insights into a single, easy-to-use interface. Users can explore **market trends, compensation benchmarks, and role-specific details** to make informed career and hiring decisions.

## Dashboard Overview
The report is split into **two main pages**:

### Page 1: High-Level Market View
Your mission control for the data job market.  
- Total job count, median salaries, and top job titles.  
- Job trends over time.  
- Comparative salary analysis (hourly vs yearly).  
- Role distribution across Data Engineers, Analysts, Scientists, and more.
